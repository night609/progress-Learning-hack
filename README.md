# Progress Learning Hack - Self-Leveling System 🌟

A gamified personal learning and skill development tracker that lets you take control of your progress journey. Award yourself stars, skip levels, and celebrate your achievements as you learn new skills.

## Features

- 🌟 **Self-Award Stars** - Recognize your own progress and accomplishments
- ⏭️ **Skip Levels** - Jump ahead when you're ready to challenge yourself
- 📊 **Progress Tracking** - Visualize your learning journey with stats and milestones
- 🎯 **Custom Learning Paths** - Create personalized skill trees
- 🏆 **Achievement Badges** - Unlock badges as you level up
- 📈 **Motivation Dashboard** - See your growth at a glance

## Why This Project?

Learning doesn't always follow a linear path. This tool empowers you to:
- Take ownership of your skill development
- Celebrate small wins with star rewards
- Break free from rigid learning structures
- Gamify your education for better motivation
- Track multiple skills simultaneously

## Getting Started

### Quick Start
1. Clone the repository
2. Open `index.html` in your web browser
3. Start tracking your learning journey!

### Usage
- **Add a Skill** - Click "Add New Skill" to create a new learning path
- **Award Stars** - Click the star icon to give yourself 1-5 stars
- **Level Up** - Earn 5 stars to automatically advance one level
- **Skip Levels** - Use the skip button when you're ready for a challenge
- **Unlock Badges** - Earn badges as you hit milestones

## Badges System

Unlock achievements as you progress:
- 🥉 Bronze Star - Reach Level 2
- 🥈 Silver Star - Reach Level 5
- 🥇 Gold Star - Reach Level 8
- 💎 Diamond - Reach Level 10
- 🔥 On Fire - Get 5 stars in a single skill
- 🚀 Rocket - Skip 3+ levels
- 👑 Master - Complete 5 different skills
